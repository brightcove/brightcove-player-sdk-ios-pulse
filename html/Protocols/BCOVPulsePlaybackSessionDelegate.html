BCOVPulsePlaybackSessionDelegate Protocol Reference

Conforms to NSObject
Declared in BCOVPulseComponent.h

Overview

A delegate protocol for users of the Brightcove Pulse advertising plugin. This delegate allows you to override the content metadata and request settings dynamically, and customize your response to clickthrough events.

– createSessionForVideo:withPulseHost:contentMetdata:requestSettings:

Called immediately before the Pulse Plugin calls OOPulse -startSessionWithDelegate: to allow the user to first modify the ads request object, for example, to change the category property. This method is optional.

- (id<OOPulseSession>)createSessionForVideo:(BCOVVideo *)video withPulseHost:(NSString *)pulseHost contentMetdata:(OOContentMetadata *)contentMetadata requestSettings:(OORequestSettings *)requestSettings
